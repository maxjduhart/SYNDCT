Convert HTML source code to a SVG image. HTML code is compressed (if over 10KB) and then set up to be injected to a foreign object (foreignObject) inline frame (Iframe). HTML is compressed to a LZMA type compression, then base64ed, then merged with a decompressor to inject to an iframe and them SVG XML code is added.

Now Gzip your SVG with 7-zip.org freeware and rename the file extension from *.svg.gz to *.svgz for even smaller results!
