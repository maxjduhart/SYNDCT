About Our JavaScript Packer:


On this page is a mashup of 10 versions (including most recent version as of Jan. 1, 2015) of our JavaScript Packer all merged into one simple to use web based application (download this page and use offline too). We have remade this packer about 20 times achieving better compression results on a test script each time. We now find that a half dozen of these versions get various results on non tests scripts, plus sometimes achieve better results than even our most current version.

Each compression algorithm gives different results; sometimes one will get better reults than another. Compression results depends on the programmer's style and layout of your source code mostly. We have tried to make default settings the best for most scripts.
