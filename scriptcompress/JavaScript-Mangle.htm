Obfuscation can never 100% protect your hard work & coding. For anyone who really wants to get at your code, it's just a speed bump. It's a waste of your money if you paid for JavaScript obfuscating services.

You can obfuscate the javascript source all you want, but it will always be reverse-engineerable just by virtue of requiring all the source code to actually run on the client machine. Often scripts are inflated in size (made bigger) with obfuscating, even this script makes your code bigger (but it will appear smaller) because it's using characters that are 2 or 3 bytes each for replacement of single byte characters. This tool is just for fun/parody/prank mostly... You can make smaller than your original obfuscated scripts on ScriptCompress.com.

Everytime this page loads, the order of replacement characters are randomized. This generator/convertor does not alter your code (other than replacing characters, it will unpack as it was originally), no comments or famatting is changed.
